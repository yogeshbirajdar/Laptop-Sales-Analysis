# Laptop-Sales-Analysis
Customer Data Processing project that enriches dataset by adding Latitude and Longitude using geocoding, with optimized performance for large datasets using Python.

## 📌 Project Overview
This project focuses on enhancing a dataset by adding **Latitude** and **Longitude** coordinates based on the `Customer_Location` column. The goal is to make the dataset ready for geographical analysis, visualization, and mapping in tools like Power BI, Tableau, or Python dashboards.

---

## 🎯 Objective
- Convert location names into geographic coordinates
- Optimize processing time for large datasets
- Prepare data for map-based analytics and insights

---

## 🛠️ Tech Stack
- Python
- Pandas
- Geopy

## ⚙️ Key Features

- Efficient handling of large datasets
